# Email

## Tech stack

- Node.js - ESM enabled
- Mali

## Environment variables

| Name | Default Value |      Description       |
| :--- | :-----------: | :--------------------: |
| PORT |     3002      | Port number to work in |

## Running

- Navigate to existing folder

```bash
cd packages/email
```

- Run on development mode and enable fast refresh

```bash
npm start
```
